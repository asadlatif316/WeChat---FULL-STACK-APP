# Chat App — MongoDB Models

## 1. User

| Field | Type | Description |
|-------|------|-------------|
| `_id` | ObjectId | Auto generated unique ID |
| `name` | String | Full name of the user |
| `email` | String | Unique email address |
| `password` | String | Bcrypt hashed password |
| `avatar` | String | Profile picture URL |
| `createdAt` | Date | Auto generated by Mongoose |
| `updatedAt` | Date | Auto generated by Mongoose |

---

## 2. Conversation

| Field | Type | Description |
|-------|------|-------------|
| `_id` | ObjectId | Auto generated unique ID |
| `participants` | [ObjectId] | Array of User refs — who is in the chat |
| `latestMessage` | ObjectId | Ref to last Message — used for sidebar preview |
| `isGroupChat` | Boolean | Default false — for future group chat support |
| `name` | String | Group name — only used when isGroupChat is true |
| `createdAt` | Date | Auto generated by Mongoose |
| `updatedAt` | Date | Auto generated by Mongoose |

---

## 3. Message

| Field | Type | Description |
|-------|------|-------------|
| `_id` | ObjectId | Auto generated unique ID |
| `conversationId` | ObjectId | Ref to Conversation this message belongs to |
| `sender` | ObjectId | Ref to User who sent the message |
| `content` | String | The text content of the message |
| `messageType` | String | `text` / `image` / `file` — default: `text` |
| `readBy` | [ObjectId] | Array of User refs who have read this message |
| `deletedFor` | [ObjectId] | Array of User refs — message hidden for these users |
| `createdAt` | Date | Auto generated by Mongoose |
| `updatedAt` | Date | Auto generated by Mongoose |

---

## Relationships

```
User ──────────────────── Conversation
         participants[]
         (many to many)

Conversation ──────────── Message
         conversationId
         (one to many)

User ──────────────────── Message
         sender
         (one to many)

User ──────────────────── Message
         readBy[]
         (many to many)
```
